Task : To find timezone related information of a place using lat + long and by address.

1. Use the api documentation given at https://www.geoapify.com/get-timezone-from-lat-long-geographical-coordinates
2. Create a Ui as shown.
3. As soon as the user comes on the web page show them their current time zone using their lat and long that you fetch using the geolocation api - https://www.w3schools.com/html/html5_geolocation.asp
4. Allow the user to type in an address and then show them the time zone at that address on the click of the button, if no time zone is shown then show them an error saying time zone could not be found.
